## About This Portfolio

This is my personal portfolio website featuring technical blog posts and projects. The site is built with Next.js and includes:

- Technical blog posts on API design, security, and development workflows
- Responsive design using modern web technologies
- GitHub integration for content management

## Featured Content

- **Workshop on REST API Design (Day 01)** - Comprehensive guide to REST API principles
- **Google Dorks & OAuth Recon Cheatsheet** - Security research tools and techniques
- **Subdomain Enumeration Workflow** - Practical reconnaissance methodology

## Technologies Used

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- Radix UI Components
- Vercel for deployment

## GitHub Integration

This portfolio features a custom blog system with GitHub integration that allows for:

- Secure authentication with JWT tokens
- Markdown-based blog editor with preview
- Automatic GitHub repository sync
- CRUD operations for blog posts

## Development

This project was initially created using v0 by Vercel.

### Local Development

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Run the development server: `pnpm dev`
